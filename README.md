# Using fbw-utils
```
npm install --save fbw-utils
```

Then,
```
var fbwUtils = require('fbw-utils')

fbwUtils.qbankFetch(...,...,...)
```

Note that you'll need the `qbank_credentials` and `handcar_credentials` file.
